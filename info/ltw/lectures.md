# LTW - Lectures & Syllabus

## Course Information
This repository contains a full 550-question database for the Web Technologies (LTW) course exam.
The question bank covers the complete syllabus for theoretical and practical assessment.

## Syllabus Topics Covered

1. **HTML (Questions 1-50)**
   - Semantic tags, document structure, forms, inputs.
   - Accessibility and SEO basics.

2. **CSS (Questions 51-100)**
   - Selectors, specificity, box model.
   - Flexbox, Grid Layout.
   - Responsive Design, Media Queries.
   - Transitions, Animations, and Variables.

3. **JavaScript Basics (Questions 101-150)**
   - Syntax, ES6+ features (let/const, arrow functions, destructuring).
   - Loops, conditionals, arrays, string manipulation.
   - Functions, Closures, Scope (Lexical environment).

4. **JavaScript DOM & AJAX (Questions 151-200)**
   - DOM manipulation, Event handling (`addEventListener`, propagation/bubbling).
   - Asynchronous JS: Callbacks, Promises, async/await.
   - XMLHttpRequest and Fetch API.
   - JSON parsing.

5. **JavaScript OOP (Questions 201-250)**
   - Prototypal inheritance, prototype chain.
   - ES6 Classes, constructors, `super()`.
   - `this` binding context (`bind`, `call`, `apply`).
   - Modules (import/export).

6. **PHP Basics & Web (Questions 251-300)**
   - Syntax, arrays, strings, dates.
   - Superglobals (`$_GET`, `$_POST`, `$_SESSION`, `$_COOKIE`).
   - File inclusion (`include`, `require`).
   - Session management and Cookies.

7. **PHP OOP (Questions 301-350)**
   - Classes, Objects, Constructors.
   - Visibility (`public`, `protected`, `private`).
   - Inheritance, Interfaces, Traits.
   - PDO (PHP Data Objects) for database interaction.

8. **Regular Expressions (Questions 351-400)**
   - Pattern matching, modifiers, character classes.
   - Quantifiers, boundaries, groupings.
   - PHP (`preg_match`, `preg_replace`) and JS (`RegExp.test`, `String.match`).

9. **Web Security (Questions 401-450)**
   - SQL Injection (Prevention via prepared statements).
   - XSS (Cross-Site Scripting) - Reflected, Stored, DOM-based.
   - CSRF (Cross-Site Request Forgery) - Anti-CSRF tokens.
   - Password hashing (`password_hash`), HTTPS, Cookies security (`HttpOnly`, `Secure`).

10. **HTTP Protocol (Questions 451-500)**
    - Request/Response cycle.
    - Methods (`GET`, `POST`, `PUT`, `DELETE`, `OPTIONS`).
    - Headers (Content-Type, Authorization, User-Agent).
    - Status Codes (200, 301, 404, 500, etc.).
    - Stateless nature and State maintenance.

11. **MPA, SPA, PWA (Questions 501-550)**
    - Multi-Page Applications (MPA) vs Single-Page Applications (SPA).
    - Client-side Routing vs Server-side Routing.
    - Progressive Web Apps (PWA) concepts.
    - Service Workers (Caching, Offline support, Background Sync, Push Notifications).
    - Web App Manifests.

## Exam Details
- **Structure**: Approximately 40 multiple-choice questions.
- **Options**: Multiple correct options may exist.
- **Penalty**: Wrong answers deduct points.
